Intelligent sensor network for controlling an irrigation system

Description

The objective of the project was to develop intelligent algorithms through which we can trigger an irrigation system according to the environmental parameters measured with help of an intelligent wireless sensors network. I created a system consisting of a sensor node, a base station and an actuator node. The sensor node is based on Arduino MKR WAN 1310 development board with LoRa communication protocol, capacitive sensor to measure soil moisture, waterproof temperature sensor, which I use to measure soil temperature and temperature and humidity sensor (DHT22/AM2302), used to determine the temperature and humidity of the air. The base station is based on Arduino MKR WAN 1310 development board with LoRa communication protocol and an OLED display which it was used to display parameters measured by sensor node. The actuator sensor is based on Arduino MKR WAN 1310 development board with LoRa communication protocol, MKR Relay Shield that can be used for loads up to 24V and a mini submersible water pump. With the help of the sensor node, the data from the sensors are read, which are transmitted to the base station where there are shown on the display. The data from the sensor node is transmitted to the actuator node at the same time. The algorithm used for the actuator node processes the data from the sensors and triggers the pump depending on the soil moisture, its temperature, but also the temperature and humidity of the air in the vicinity of the plant.
